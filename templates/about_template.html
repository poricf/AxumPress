About This Project

Axumpress is a static site generator built in Rust, designed for speed, safety, and modern web standards.
It uses Askama for templating and Comrak for Markdown parsing.

Features:

  • Write posts in Markdown
  • Customizable themes and templates
  • Fast static HTML generation
  • Easy deployment to any static host

Source code: github.com/poricf/axumpress
